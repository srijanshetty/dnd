Elementary OS do not disturb mode. This will kill chrome and start the do not disturb mode
for OS notifications.

Install
-------

1. Copy dnd to anywhere in \$path
2. For completions, copy \_dnd to some locations in \$fpath.

USAGE
------

```
USAGE: dnd | dnd off
```
